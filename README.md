
# What it does

This tool is designed to allow you to automatically view a site on different screen sizes and browsers. This is useful for checking accessibility and frontend design on your website
Current options are widths of 1024, 768 and 402 with automatic height, this is meant to closely emulate a desktop, tablet and mobile.
The default browsers used are Google Chrome, Microsoft Edge and Firefox (unfortunately safari is not available without a mac).
All these options can be easily modified via the comfig options in the main py file
